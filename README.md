# daochan-backend

The backend monorepo for the Daochan platform

## Prerequisites

- Install Go
- Install Docker
- Install Make
- Install sqlc `go install github.com/kyleconroy/sqlc/cmd/sqlc@latest`
- Install goose `go install github.com/pressly/goose/v3/cmd/goose@latest`
